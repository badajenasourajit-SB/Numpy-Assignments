NumPy Assignment

Name: Sourajit Badajena

Description

This repository contains solutions to the first six NumPy assignment questions. The programs demonstrate array creation, indexing, slicing, reshaping, arithmetic operations, and random number generation using the NumPy library.

Questions Covered

Question 1

- Extract first column
- Extract last row
- Extract specific sub-matrix from a 3×3 array

Question 2

- Extract first row
- Extract second column
- Extract a specific element from a NumPy array

Question 3

- Create an array from 1 to 12
- Reshape into:
  - (3,4)
  - (2,6)
  - (2,3,2)

Question 4

Perform operations on NumPy arrays:

- Addition
- Subtraction
- Multiplication
- Division
- Power
- Square
- Mean

Question 5

- Generate a 1D random array
- Generate a 2D random array of shape (3,3)

Question 6

- Generate a random array of shape (2,5)

Requirements

- Python 3.11
- NumPy
